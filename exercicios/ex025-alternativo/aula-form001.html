<form>

  • Elemento que definirá um formulário
  • É um container estilo <section> e <footer>

Atributos básicos:

  • action
  • method

O elemento <form> define um formulário. É um elemento de container como um elemento <div> ou <p>, mas ele também suporta alguns atributos específicos para configurar a forma como o formulário se comporta. Todos os seus atributos são opcionais, mas é considerada a melhor prática sempre definir pelo menos o atributo action e o atributo method.

  • O atributo action define o local (uma URL) em que os dados recolhidos do formulário devem ser enviados.
  • O atributo method define qual o método HTTP para enviar os dados (ele pode ser "GET" ou "POST" (O method GET mostra as informações na URL do navegador e o method POST oculta essas informações. Se usar o GET a URL do site vai ficar assim quando for cadastrando os dados no formulário: https://minhapaginaweb.com.br/?name=Guilherme&email=guilherme@algumacoisa.com)
