ELEMENTO <fieldset>

O  elemento HTML <fieldset> é usado para agrupar elementos, assim como <label>, dentro de um formulário web.

O <fieldset> é usando como para agrupamento de campos com o mesmo propósito. Esse elemento é um elemento semântico que ajuda na indexação do site e da mais acessibilidade aos usuários já que o <fieldset> é melhor interpretado pelos leitores de tela. Por padrão ele tem um recurso visual que é uma linha em volta do conteúdo.

ATRIBUTOS

Este elemento inclui os atributos globais. (Atributos globais são atributos comuns a todos os elementos HTML; eles podem ser usados em todos os elementos, embora possam não ter efeito em alguns elementos).

  • disabled: O atributo `disabled` desabilita todos os elementos internos. E não será enviado dados ao submeter o formulário. O `disabled` é um atributo booleano, ou seja, para usa-lo basta colocar no código, se não precisar dele basta não coloca-lo.
  • form: O atributo `form` serve como o id de um formulário ao qual esse `fieldset` pertence. Não precisa estar dentro do formulário. Seu valor padrão é o **id** do elemento `[
    ](https://developer.mozilla.org/pt-BR/docs/Web/HTML/Element/form)` mais próximo servindo como um link entre um e outro.
  • name: O atributo `name` serve para dar nome ao grupo.
  • legend: O atributo `legend` serve para dar nome do agrupamento e é o primeiro elemento dentro do `fieldset`.
