ELEMENTO <button>

O elemento <button> Representa um botão que tem algumas formas para ser usados, como por exemplo, é usado para enviar formulários. É estilizado pelo user agent, ou seja, cada navegador (user agent) vai estilizador do seu próprio jeito, podendo ter seu estilo mudado pelas CSS.

ATRIBUTOS

  • type
    • submit: um botão com o type=”submit”* é usado para enviar o formulário.
    • reset: um botão com o type=”reset” vai limpar os campos de formulário.
    • button: um botão com o type=”button” é usado para enviar o formulário também.
  • autofocus: o atributo autofocus deixa o elemento marcado assim que abre a página.
  • disabled: o atributo disabled desativa a funcionalidade do elemento.
  • name
  • value
  • form
            
                <form action="" method="">
                    <label for="nome">Nome Completo</label>
                    <input id="nome" type-text>
                    <button type="submit">Enviar</button>
                </form>

O type poderia ser button que faria a mesma coisa, que é enviar o formulário. Lembrando que para a gente ainda não aprendeu como receber os dados do formulário. Então mesmo o botão sendo clicado não vamos receber.

RESET

Visualmente não muda a estética do botão. Mas agora ao ser clicado ele apaga o campo.

            
                <form action="">
                    <label for="nome">Nome Completo</label>
                    <input id="nome" type-text>
                    <button type="reset">Limpar</button>
                </form>

AUTOFOCUS

            
                <form action="">
                    <label for="nome">Nome Completo</label>
                    <input id="nome" type-text>
                    <button type="submit" autofocus>Enviar</button>
                </form>

Note que o botão está com uma borda preta. Isso porquer o botão está com o autofocus e ele já estar selecionado. Se clicar enter no teclado ele já enviar o formulário vazio

DISABLED

É bom usar o disabled e juntar ele com o javascript para não deixar o usuário enviar o formulário até que todos os campos estejam preenchidos.

            
                <form action="">
                    <label for="nome">Nome Completo</label>
                    <input id="nome" type-text>
                    <button type="submit" disabled>Enviar</button>
                </form>

NAME & VALUE

o name pode ser usado junto com o value. O atributo dentro do value será recebido em algum lugar com o name button. Isso só vai fazer sentido quando aprendemos a receber o dados.

            
                <form action="">
                    <label for="nome">Nome Completo</label>
                    <input id="nome" type-text>
                    <button type="submit" name="button" value="valor_qualquer">Enviar</button>
                </form>

FORM

            
                <form action="" method="" id="meu-formulario"></form>
                <button type="submit" form="meu-formulario"></button>

Quando esse ↑ botão for clicado, os dados que estiverem na tag form será enviado. O atributo form é usado para linkar os elementos. Se a tag form estivessem em um local distante de um botão é só criar esses link de id e form.
