ELEMENTO <button>

O elemento <button> Representa um botão que tem algumas formas para ser usados, como por exemplo, é usado para enviar formulários. É estilizado pelo user agent, ou seja, cada navegador (user agent) vai estilizador do seu próprio jeito, podendo ter seu estilo mudado pelas CSS.

ATRIBUTOS

  • type
    • submit: um botão com o type=”submit”* é usado para enviar o formulário.
    • reset: um botão com o type=”reset” vai limpar os campos de formulário.
    • button: um botão com o type=”button” é usado para enviar o formulário também.
  • autofocus: o atributo autofocus deixa o elemento marcado assim que abre a página.
  • disabled: o atributo disabled desativa a funcionalidade do elemento.
  • name
  • value
  • form
